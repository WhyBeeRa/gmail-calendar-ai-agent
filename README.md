# AI Agent - Auto Meeting Scheduler 📅🤖

ברוכים הבאים לפרויקט הסוכן האוטונומי לניהול פגישות!
פרויקט זה פותח במסגרת קורס "סוכני בינה מלאכותית" של ד"ר יורם סגל.

## תיאור המערכת
הסוכן סורק תיבת Gmail, מזהה זימונים לפגישות הכתובים בשפה טבעית, בודק זמינות ב-Google Calendar, וקובע אירועים או דוחה אותם באופן אוטומטי לחלוטין.

## איך להתחיל (Getting Started)

### דרישות קדם
1. **Python 3.10 ומעלה** מותקן על המחשב.
2. **חשבון Google Cloud Console** עם Gmail API ו-Google Calendar API מופעלים.
3. הורדת קובץ ה-Credentials (מסוג OAuth 2.0 Client ID) מ-Google Cloud ושמירתו בתיקיית הפרויקט כ-`client_secret_*.json` (או `credentials.json`).

### התקנה
1. פתח טרמינל בתיקיית הפרויקט.
2. צור סביבה וירטואלית והפעל אותה:
   ```powershell
   python -m venv .venv
   .venv\Scripts\activate
   ```
3. התקן את התלויות הדרושות:
   ```powershell
   pip install -r requirements.txt
   ```

### הגדרות (Configuration)
1. צור קובץ `.env` בתיקיית השורש והגדר את מפתח ה-API של Gemini:
   ```env
   GEMINI_API_KEY=your_actual_api_key_here
   ```
2. ודא שקובץ ה-`client_secret_*.json` שקיבלת מ-Google Cloud Console נמצא בתיקיית השורש.

### הרצה
הרז את לולאת העבודה של הסוכן:
```powershell
python main.py
```
בהרצה הראשונה ייפתח חלון בדפדפן לצורך מתן הרשאה (OAuth 2.0) לחשבון הגוגל שלך. לאחר האישור, ייווצר קובץ `token.json` המאפשר גישה אוטומטית בהרצות הבאות.

## זרימת העבודה של הסוכן (Flow Chart)
1. **קריאה (Read):** הסוכן סורק אימיילים שלא נקראו בתיבת ה-Gmail שלך.
2. **ניתוח (Analyze):** תוכן כל אימייל נשלח ל-Gemini 2.5 Flash שמנתח בשפה טבעית האם מדובר בבקשת פגישה.
   - אם מדובר בזימון, הוא מחלץ: נושא, תאריך ושעה (ISO 8601), משתתפים ותיאור.
3. **בדיקת זמינות (Check Availability):** הסוכן בודק ב-Google Calendar אם טווח הזמן פנוי.
4. **ביצוע פעולה (Execute):**
   - **אם פנוי:** קובע אירוע ביומן ושולח מייל אישור עם קישור לפגישה.
   - **אם תפוס:** שולח מייל סירוב מנומס המבקש מועד חלופי.
   - **אם חסר מידע:** שולח מייל בקשת הבהרה (לדוגמה אם חסרה שעה או תאריך).
   - בכל מקרה, האימייל מסומן כנקרא (`Read`) כדי למנוע כפילויות.
